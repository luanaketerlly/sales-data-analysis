# Análise de Dados de Vendas

## 📊 Visão Geral do Projeto

Este projeto tem como objetivo realizar a limpeza e a análise exploratória de um dataset financeiro de vendas.

A proposta é compreender o comportamento das vendas, lucro e descontos, além de identificar possíveis padrões relacionados a produtos, segmentos e países.

Este projeto faz parte do meu processo de aprendizado em **Análise de Dados** e continuará evoluindo conforme o avanço da minha mentoria e estudos.

---

## 📁 Dataset

O dataset utilizado contém informações financeiras relacionadas a vendas, incluindo colunas como:

* Segment
* Country
* Product
* Discount Band
* Units Sold
* Price
* Sale Price
* Gross Sales
* Discounts
* Sales
* Profit
* Date

Esses dados permitem analisar o desempenho das vendas e explorar possíveis insights para tomada de decisão.

---

## 🧹 Tratamento de Dados

Durante a preparação dos dados foram realizadas algumas etapas de limpeza e transformação, como:

* Remoção de símbolos monetários e formatações
* Conversão de colunas financeiras para formato numérico
* Tratamento de valores inconsistentes
* Criação de colunas auxiliares relacionadas ao tempo

Essas etapas são importantes para garantir que os dados estejam adequados para análise.

---

## 🔎 Análise Exploratória de Dados

A análise inicial inclui:

* Inspeção da estrutura do dataset
* Estatísticas descritivas
* Verificação de possíveis inconsistências
* Primeiras transformações nos dados

Novas análises e visualizações serão adicionadas conforme o desenvolvimento do projeto.
